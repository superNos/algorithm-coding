
## 斐波那契数列
**题目描述**

大家都知道斐波那契数列，现在要求输入一个整数n，请你输出斐波那契数列的第n项（从0开始，第0项为0）。<br>
n<=39

```javascript
    function Fibonacci(n)
    {
        let array = [0, 1]
        if(n < 2) return array[n]
        let i = 1
        while(++i <= n) {
            array[i] = array[i - 1] + array[i - 2]
        }
        return array[n]
    }
```

**思路**
斐波那契数列，su`第n项 = 第n-1项 + 第n-2项`
